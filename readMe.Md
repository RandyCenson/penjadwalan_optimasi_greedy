# Sistem Penjadwalan Seminar Magang FTI UNTAR

Pendekatan Greedy untuk Optimasi Penjadwalan Seminar Magang

## Deskripsi Singkat

Aplikasi ini merupakan sistem penjadwalan seminar magang berbasis web yang dirancang untuk membantu panitia Fakultas Teknologi Informasi Universitas Tarumanagara dalam menyusun jadwal seminar magang secara otomatis. Sistem ini menggunakan algoritma Greedy sebagai pendekatan heuristik untuk menghasilkan jadwal yang valid, bebas konflik, dan efisien, serta menyediakan fitur perbandingan dengan model matematis dan data penjadwalan manual.

---

## Persyaratan Sistem

Pastikan lingkungan pengembangan telah memenuhi kebutuhan berikut:

-   Python versi 3.10.0
-   Node.js dan npm
-   Browser modern

---

## Instalasi dan Menjalankan Aplikasi

### 1. Instal Python

Unduh dan instal Python versi 3.10.0 dari situs resmi Python.

### 2. Instal Dependensi

Jalankan perintah berikut pada direktori proyek:

```
npm install
pip install -r requirements.txt
```

### 3. Menjalankan Aplikasi

Untuk menjalankan aplikasi web, gunakan perintah:

```
npm run dev
```

Setelah aplikasi berjalan, sistem dapat diakses melalui browser.

## Fitur Utama

-   Ekspor hasil penjadwalan ke format Excel

## Catatan Penggunaan

-   Format file Excel mahasiswa dan preferensi dosen harus sesuai dengan spesifikasi yang telah ditentukan.
-   Hasil penjadwalan dapat disesuaikan kembali secara manual jika diperlukan.
